My research portfolio focuses on translating chaotic event streams into resilient robotic spatial awareness. Future Research & Existing Work

Event Vision · Log-Polar

Log-Polar Equivariant Features

Extracting rotation- and scale-equivariant features from asynchronous event streams via log-polar filtering, enabling robust ego-motion estimation under heavy camera spin and forward translation.

Live Demo →
VSA · Cognitive Mapping

VSA Cognitive Mapping

Binding log-polar event features into superposed hypervectors (FHRR) to build compact, queryable spatial maps on edge hardware — bridging continuous neuromorphic perception with discrete symbolic reasoning.

Try the VSA Explorer →
Uncertainty · Active Sensing

Uncertainty-Driven Active Perception

A continuous epistemic uncertainty monitor over VSA resonance directs the robot's attention — triggering SPAD or LiDAR fallback only when the scene exceeds ambiguity thresholds, realising sense-less-understand-more efficiency.

See Active Fallback →
Neuromorphic · Structural Plasticity

Structicity

Structural plasticity mechanisms for spiking neural networks — dynamic rewiring of synaptic connectivity in response to sensory statistics, enabling continual adaptation without catastrophic forgetting.

View Publications →
SNN · Event Vision · Segmentation

Event Camera SNN Segmentation

First unsupervised semantic and instance segmentation spiking neural networks operating directly on DVS event data — no frame conversion, no labelled training set.

View Publications →
SPAD · 3D Imaging · Asynchronous

Event-Based SPAD Imaging

First asynchronous 3D imaging pipeline coupling event cameras with SPAD arrays — exploiting photon-counting sensitivity and microsecond timing for depth reconstruction in low-light autonomous navigation.
