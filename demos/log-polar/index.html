Log-Polar Retina — Spin & Forward Motion Demo

A corridor scene viewed through a moving camera, with a configurable log-polar kernel map on the right. Forward motion produces mainly vertical drift in log-polar space; camera spin produces mainly horizontal drift. The right panel can show a continuous transform or a discrete retinal kernel mosaic — receptor, bipolar (ON/OFF), or ganglion-like spiking stages.

1. Continuous Easiest geometric reading. Transformed feature points, motion trails, and flow vectors are plotted directly.
2. Kernel mosaic Features are pooled into discrete log-polar bins with uniform or Gaussian kernels — closer to a real retinal sampling arrangement.
3. Cell-stage views Receptor, bipolar, and ganglion-style panels approximate retinal pipeline stages. Browser versions — not exact parity with Python internals.

Geometry vs retinal sampling
Continuous mode is best for intuition. Mosaic and cell-stage modes are closer to the structure of a retina-inspired kernel codebase.
The Pyrception repository advertises a log-polar kernel arrangement, uniform or Gaussian kernels, arbitrary kernel size/shape/orientation, major retinal cell types, temporal dynamics, events as input, and saccadic movements. This browser version mirrors that structure visually but is a hand-built JavaScript approximation rather than a direct execution of the Python package.
